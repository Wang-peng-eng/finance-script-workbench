# 财经短视频选题—口播工作台

## 明天怎么用

1. 双击 `index.html`。
2. 填写热点关键词，粘贴参考资料和来源链接。
3. 点击“复制选题提示词”，发送给你平时使用的 AI。
4. 把 AI 返回的 10 条选题粘回工作台，点击“解析并评分”。
5. 从 TOP3 中人工选择最终选题。
6. 点击“复制口播提示词”，发送给 AI。
7. 把返回的纯口播稿粘回工作台。
8. 检查字数、预计时长、数字项和风险词。
9. 点击“导出 TXT”。

## 无法调用 AI 时

可以使用“生成10个本地选题”和“生成本地结构草稿”。本地模式不联网，不会读取实时数据，适合作为结构草稿，不建议未经核验直接发布。

## 数据

当前任务自动保存在浏览器本地，不会上传。点击“新建任务”会清空当前任务。
